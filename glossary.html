Glossary

Definitions for common terms used on this site.

Client-Side

Refers to operations that are performed by the user's browser (the "client") on their own computer, rather than on a remote server. This entire website runs client-side, which means your data is not sent to us.

GUI (Graphical User Interface)

A type of user interface that allows users to interact with electronic devices through graphical icons and visual indicators, as opposed to text-based command-line interfaces. Stacher is a GUI for yt-dlp.

Open Source

Software for which the original source code is made freely available and may be redistributed and modified. Sinny's Palace is an open-source project, and you can view its code on GitHub.

WAV (Waveform Audio File Format)

An uncompressed audio file format. Because it is lossless, it maintains the highest possible audio quality, which makes it ideal for music production and professional audio editing.

yt-dlp

A command-line program to download videos from YouTube and many other sites. It is a powerful and popular fork of the original youtube-dl project.
